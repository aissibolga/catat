Pertama, install dulu Ruby.

Kalau menggunakan Linux:

```bash
sudo apt install ruby
```

Kalau menggunakan Termux:

```bash
pkg install ruby
```

Kemudian, install Jekyll dengan perintah:

```bash
gem install jekyll
```
